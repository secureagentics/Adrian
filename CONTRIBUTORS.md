## How to add yourself

Open a pull request adding your name to the relevant section, in alphabetical order within that section. A one-line entry is fine:

```
- Your Name (@your-github-handle) - one-line description of what you worked on
```

If your contribution is not code (docs, design, testing, an issue report we acted on, a bug repro), say so in the description. Anything that materially shaped the project is worth listing.
